/*----API----*/
Esta es una prueba de conocimiento para la vacante QA

Para poder ejecutar las pruebas del API debemos realizar:
1. Descargar el la coleccion adjunta (Bryan Armas - BP.postman_collection.json)
2. Abrir POSTMAN
3. Menú/File/Import
4. Seleccionar el archivo descargado
5. Ejecutar las pruebas con los métodos POST (Crear Usuario), GET (Buscar Usuario), PUT (Actualizar Usuario)
, DELETE (Eliminar Usuario)
6. Visualizar los resultados

NOTA: Esta colección se creó en base a la documentación brindada en swagger, viendo el Endpoint, parámetros y 
estructuras JSON solicitados en cada método
